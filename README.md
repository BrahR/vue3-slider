## Installation 
```js
# npm
npm install vue3-slider
```

```js
# yarn 
yarn add vue3-slider
```

## Getting Started 👍

### Import component
```js
import slider from "vue3-slider"

export default {
  ...
  components: {
    "vue3-slider": slider
  }
  ...
}
```

### How to use
```html
<vue3-slider
  v-model="myNumber"
  color="#FB278D"
  track-color="#FEFEFE"
>
```
